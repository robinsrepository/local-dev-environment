Local Dev Environment

Text Editor

To edit your code on your pc, I do this with Visual Studio Code.

Version Control

I use it to keep changes updated on GitHub, using GitHub Desktop.

Commit

GitHub makes an official record of your changes when you commit them. Don't forget to save your changes in VSC before you do this. By clicking the "History" tab in GitHub Desktop, you can view all your commits.

Push

GitHub publishes the changes in your files and commits you've made on GitHub when you push them.

Deploy

Once you've pushed your changes to GitHub, you can make your code available on a server so that others can view the website online. I do this with GitHub Pages: my code is uploaded to a special place where GitHub automatically converts it into a working website.
